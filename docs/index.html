Everything you copy, instantly findable

Stash quietly records your clipboard and indexes it with SQLite FTS5, so search never slows down — no matter how much history you accumulate.

📋

Live capture

Every text clip is saved automatically, tagged with the source app and time — with the app's real icon shown on each entry.

⚡

Three search modes

Substring "contains", relevance-ranked word search, and full regular expressions — switch on the fly.

🕘

Recent first

Opens to your latest clips. Start typing to search; clear to go back to recent.

🖼️

Images and GIFs

Copied images are captured as thumbnails; GIFs show a still first frame. PNG, JPEG, WebP, AVIF, HEIC and more, each labeled by format.

🔁

Smart duplicates

Re-copying something already saved moves it to the top instead of duplicating — or keep every copy if you prefer.

🖱️

One-click copy

Click a result or press Return to copy it and close. Paste with ⌘V anywhere.

⌨️

Customizable hotkey

Summon the search bar from any app. Default is ⌃⌥C, and you can record your own.

✨

AI regex (opt-in)

Add an OpenCode API key and describe a pattern in plain English — a free model writes the regex and runs the search.

🔤

Copy transformations

Upper/lower/capitalize, trim or strip whitespace, single-line, or prepend/append text — applied as you copy.

⭐

Favorites and cleanup

Right-click any result to favorite or delete it. Filter the view to just your starred clips.

🪟

Compact or centered

Open as a centered Spotlight-style window, or a compact dropdown anchored under the menu-bar icon.

🎨

Themes

Six dark looks — pick yours in Settings.

💾

Export and import

Save your whole history to a standalone SQLite database, or import once from a Copy 'Em export, strictly read-only.

Get started in seconds

Download & open the disk image, then drag Stash onto Applications.
Launch it from Applications or Spotlight — a clipboard icon appears in your menu bar.
Press ⌃⌥C (or left-click the icon) and start searching your clips.

Build from source

Requires the Swift toolchain (ships with Xcode).

git clone https://github.com/Alyetama/Stash.git Stash && cd Stash
./build.sh        # builds + installs to /Applications
./make-dmg.sh     # builds + packages dist/Stash.dmg
