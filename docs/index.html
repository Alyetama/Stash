Everything you copy, instantly findable

Stash quietly records your clipboard and indexes it with SQLite FTS5, so search never slows down — no matter how much history you accumulate.

📋

Live capture

Every text clip is saved automatically, tagged with the app it came from and when.

⚡

Three search modes

Substring "contains", relevance-ranked word search, and full regular expressions.

🕘

Recent first

Opens to your latest clips. Start typing to search; clear to go back to recent.

🖱️

One-click copy

Click a result or press Return to copy it and close. Paste with ⌘V anywhere.

⌨️

Global hotkey

Summon the search bar from any app with ⌃⌥⌘C. No Dock icon, no clutter.

📦

Copy 'Em import

Already use Copy 'Em? Your history is imported once, strictly read-only.

Get started in seconds

Download & open the disk image, then drag Stash onto Applications.
Launch it from Applications or Spotlight — a magnifier icon appears in your menu bar.
Press ⌃⌥⌘C (or left-click the icon) and start searching your clips.
⚠️ Stash is a clipboard manager, so it records everything you copy — including passwords and tokens. App-exclusion and a pause toggle are on the roadmap. Everything stays on your Mac; Stash has no network access.

Build from source

Requires the Swift toolchain (ships with Xcode).

git clone <your-repo-url> Stash && cd Stash
./build.sh        # builds + installs to /Applications
./make-dmg.sh     # builds + packages dist/Stash.dmg
